# CTA Director

CTA Director is an AI-powered creative assistant for generating
production-ready OpenAI Flow prompts for high-converting ecommerce CTA
end cards.

Upload a Website Screenshot or Product Image, let the AI extract key
information, customize the layout and style, and instantly generate a
premium Flow prompt optimized for mobile-first advertising.

## ✨ Features

-   📸 Upload Website Screenshots or Product Images
-   🤖 AI-powered extraction of headlines, product names, prices,
    reviews, benefits, brand colors and trust elements
-   🎨 Multiple visual styles and layout inspirations
-   🧩 Interactive CTA Layout Builder
-   📱 Instagram / Reels safe area visualization
-   ✏️ Editable AI-detected content
-   🔄 Regenerate AI copy suggestions
-   📦 Exact packaging and branding preservation
-   🎯 Mobile-first ecommerce optimization

## Workflow

1.  Upload a Website Screenshot or Product Image.
2.  Let the AI analyze the asset.
3.  Review and edit detected content.
4.  Choose a visual style and layout inspiration.
5.  Apply changes.
6.  Copy the generated OpenAI Flow prompt.
7.  Attach the original reference image in Flow and generate your CTA.
